# Horsey Game Monitor

Horsey Game 专用赛马数据分析器。一个基于 Python + Tkinter 的桌面工具，采用**记录与统计分离**的架构，用于管理马、记录比赛数据、分析统计数据，并可视化血缘谱系。

![界面截图](window.png)

## 核心架构：记录与统计分离

本工具将数据模型分为两层：

| 层级 | 作用 | 示例 |
|---|---|---|
| **记录列** | 定义添加记录时可录入的原始字段 | 1线(time)、2线(time)、摔倒(bool) |
| **统计列** | 定义主表格中显示的统计指标，引用记录列 | 1线平均(Avg of col_1)、1线最大(Max of col_1) |

**优势**：
- 同一记录列可以衍生出多个统计列（如同时看平均、最大、最小值）
- 统计列名称完全自由定义
- 记录列的增删不影响其他记录列的历史数据

### 支持的记录列类型

| 类型 | 说明 | 添加记录时控件 |
|---|---|---|
| `time` | 时间/秒数 | Entry + 录入时间按钮 + 清空按钮 |
| `range(a,b)` | a 到 b 的整数 | Spinbox 整数选择器 |
| `bool` | 是/否 | Checkbutton 勾选框 |

> 所有类型底层都是数字，保存时统一校验。

### 支持的统计方式

| 方式 | 说明 | 显示格式 |
|---|---|---|
| `avg` | 平均值 | `7.09 (4)` |
| `max` | 最大值 | `10.20 (4)` |
| `min` | 最小值 | `1.20 (4)` |
| `sum` | 累计值 | `42.50 (4)` |

括号内为样本量。

## 功能特性

### 🐴 马管理
- **添加马**：支持随机生成英文名（自动去重、限制7字符内）、一键复制名称
- **编辑马**：随时修改马的名字、父亲、母亲
- **删除马**：删除马及其所有历史记录
- **隐藏马**：标记不活跃的马，可控制是否在列表中显示
- **ID 自增**：自动分配唯一 ID

### ⏱️ 记录系统
- **添加记录窗口**：只显示记录列（由"记录管理"定义）
  - `time` 类型：一键录入计时器时间、手动编辑、清空
  - `range` 类型：Spinbox 选择范围内的整数
  - `bool` 类型：勾选框
- **数据校验**：保存时自动检查所有值必须是合法数字

### 📊 统计系统
- **完全自定义统计列**：通过"统计管理"自由添加、编辑、删除统计列
- 每个统计列指定：名称、来源记录列、统计方式(avg/max/min/sum)
- 主表格实时显示所有统计列
- 支持点击列头升序/降序排序
- **共子**：未选中时显示自己的子女数；选中某匹马时，显示与该马共同育有的子女数

### 🩸 血缘可视化
- 选中某匹马时：
  - **祖先**淡蓝色高亮（越近越深）
  - **后代**淡粉色高亮（越近越深）
  - 自身黄色高亮
- 共子列同步显示共同子女数量
- 支持"仅显示亲缘马"过滤，快速聚焦血统关系
- 再次点击已选中的行可取消选中

### 📝 记录明细
- 选中马后下方自动显示该马所有历史记录
- bool 类型值为 1 时显示 ✓

### 💾 数据导入导出
- **导出**：保存为带时间戳的 ZIP，包含 `horses.csv`、`records.csv`、`record_schema.csv`、`stat_schema.csv`
- **导入**：选择 ZIP 文件覆盖现有数据
- **清空**：一键清空所有数据（不可恢复）

## 运行方式

### 环境要求
- Python 3.10+
- `tkinter`（Python 内置）
- `faker`

### 安装依赖
```bash
pip install faker
```

### 启动
```bash
python horse_racing.py
```

数据文件会自动生成在当前目录。

## 数据文件说明

| 文件 | 内容 |
|---|---|
| `horses.csv` | 马基础信息：ID、名字、父亲 ID、母亲 ID、添加日期、隐藏状态 |
| `records.csv` | 原始记录：记录 ID、马 ID、马名、日期、以及记录列定义的动态列 |
| `record_schema.csv` | 记录列定义：列ID、名称、类型(time/range/bool)、参数、顺序 |
| `stat_schema.csv` | 统计列定义：统计ID、名称、来源列、统计方式、顺序 |

## 界面布局

- **上半区**：马数据统计表格（由统计列定义）
- **下半区**：选中马的记录明细（由记录列定义）
- **顶部工具栏**：添加 / 编辑 / 删除 / 添加记录 / 刷新 / 记录管理 / 统计管理 / 显示隐藏 / 仅显示亲缘 / 数据操作

## 提示

- 隐藏马不会出现在父母下拉框中
- 删除记录列时会同时删除引用它的所有统计列
- 导入旧版备份时会自动迁移到新格式
